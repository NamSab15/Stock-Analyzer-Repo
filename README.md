# Indian Stocks Sentiment Analyzer

This repository contains a full-stack app for monitoring Indian stocks with sentiment analysis, real-time updates, and charts.

Contents
- backend: Node/Express backend
- frontend: React frontend with Vite
- Documentation and deployment guides

To get started locally, use the commands in `DOCUMENTATION_INDEX.md`.

Note: Do not store secrets in this repository. Use `.env` files and configure env vars in your deployments.
